ABOUT $PCLUB

Pepe's Club uses a deflationary mechanism in which a small percentage of tokens gets burnt with each transaction, creating scarcity and increasing the value of the remaining tokens over time. Additionally, it employs a redistribution system, where a portion of every transaction is shared amongst the existing token holders, fostering user engagement and long-term investments
